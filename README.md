# TodoMaster - Simple Task Manager

A clean, responsive todo list application that works on both desktop and mobile devices.

## Features
- ✅ Add, complete, and delete tasks
- ⭐ Mark important tasks
- 📅 Filter by today, week, completed
- 🏷️ Categorize tasks (Work, Personal, Shopping)
- 🔄 Sort tasks by priority
- 📱 Fully responsive design

## How to Use
1. Open `index.html` in any modern browser
2. Add tasks using the input field
3. Click checkbox to mark as complete
4. Use filters to organize your view
5. Click the star to mark important

## Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- Vanilla JavaScript (No frameworks!)
- Font Awesome Icons

## Browser Support
- Chrome, Firefox, Safari, Edge (latest versions)
- Mobile browsers (iOS Safari, Chrome for Android)
